Frequently Asked Questions

Everything you need to know about TooltipAI.

General

What is TooltipAI?
TooltipAI is a Windows desktop application that overlays smart, AI-powered tooltips on top of any UI element you hover over. It uses Windows UI Automation (UIA) to detect elements and generates contextual explanations using AI. It works with 100+ apps including Ableton Live, VS Code, Chrome, Blender, and more.
Is TooltipAI free?
Yes! TooltipAI has a generous free tier that includes UIA element detection, basic tooltips, keyboard shortcuts, and multi-monitor support. Pro ($4.99/mo) adds AI-powered context, history, bookmarks, and custom themes. Business ($9.99/mo) includes API access and dedicated support.
Which Windows versions are supported?
TooltipAI supports Windows 10 (version 1903 and later) and Windows 11. Both x86-64 processors are required.
Does TooltipAI slow down my computer?
TooltipAI is designed to be lightweight. It typically uses under 2% CPU and less than 30MB of RAM. It uses adaptive throttling — polling at 50ms when active and 200ms when idle — to minimize resource usage.

Installation & Setup

How do I install TooltipAI?
TooltipAI is portable — no installer needed. Download the ZIP from our releases page, extract it to any folder, and run TooltipAI.exe. See the Getting Started guide for detailed instructions.
Why does my antivirus flag TooltipAI?
TooltipAI uses the Windows UI Automation API to read screen elements, which some antivirus tools interpret as suspicious behavior. This is a false positive — TooltipAI does not modify, inject into, or interact with other applications. You can safely add it to your exclusion list.
Do I need admin rights to run TooltipAI?
No. TooltipAI runs as a regular user process. However, if you want tooltips for elevated (Administrator) windows, you'll need to run TooltipAI as Administrator as well.

Features & Usage

Which apps are supported?
TooltipAI supports 100+ applications including Ableton Live, FL Studio, VS Code, Chrome, Firefox, Blender, Photoshop, Premiere Pro, Audacity, Microsoft Word, Windows Terminal, and many more. Any application that exposes UI elements via the Windows UI Automation API will work with TooltipAI.
How does the AI context work?
When you hover over an element, TooltipAI reads its UIA properties (name, type, role, value) and sends them to an AI model to generate a contextual explanation. In offline mode, it falls back to built-in rule-based tooltips. Pro users can use their own OpenAI API key for more detailed responses.
What are category themes?
TooltipAI color-codes tooltips by software category: Audio (blue), Creative (pink), Development (green), Browser (yellow), and Terminal (purple). This makes it easy to identify the context at a glance. Custom themes are available for Pro users.
Can I use TooltipAI with multiple monitors?
Yes! TooltipAI fully supports multi-monitor setups. Tooltips adapt to screen edges to remain visible and properly positioned regardless of which display you're using.

Privacy & Security

Does TooltipAI collect my data?
TooltipAI sends UIA element data (element name, type, and role) to the AI service to generate contextual explanations. No personal files, screenshots, or browsing history are collected. Your OpenAI API key (if provided) stays on your device and is never sent to our servers. See our Privacy Policy for full details.
Is TooltipAI GDPR compliant?
Yes. TooltipAI is fully GDPR compliant. We process data lawfully, collect only what's necessary, and respect your rights to access, rectify, and delete your data. See our Privacy Policy for complete information.
Is my OpenAI API key safe?
Yes. Your API key is stored locally on your device and is only used to communicate directly with OpenAI's API. It is never transmitted to TooltipAI's servers. You can remove it at any time from Settings.
